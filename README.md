## slot  文件插槽 
当父组件自定义标签 内部有内容的时候  需要讲内容分发到子级的内部  
### ［1］具名插槽  
有名字的插槽 父级标签的 slot属性 ＝"name" 自己模版的slot标签 的name 属性＝“name” 内容被分发到不同的name 标签中 
### ［2］ 作用域插槽   
父级别中有特殊的slot－scope 的 <template>元素   slot-scope 的值将被用作为一个临时变量名 该值从子组件传递过来的prop  对象
