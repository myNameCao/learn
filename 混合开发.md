* 2  通过 WebView的 evaluateJavascript（）  ***可以获得返回值***

```js 

  mWebView.evaluateJavascript（"javascript:getShareContent(" + shareType + ")", new ValueCallback<String>() {
        @Override
        public void onReceiveValue(String value) {
            //此处为 js 返回的结果
        }
    });
```
